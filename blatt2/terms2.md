# Vertex

Vertices sind die Sammlung von Daten für 3D Koordinaten

# Fragment

Die Daten, die OpenGL zum Rendern eines Pixels benötigt, werden als Fragment bezeichnet.

# Shader

Shaders sind die kleine Programs, die im GPU ausgeführt werden.

# Normalized Device Coordinates

Normalisierte Gerätekoordinaten sind ein kleiner Raum, in dem die x-, y- und z-Werte von -1,0 bis 1,0 reichen.

# OpenGL Object

Ein Object ist eine Sammlung von OpenGL Optionen

# Vertex Buffer Object

VBO sind die Object, die uns ermöglicht, mehr als ein Vertex in GPU zu speichern. Es managiert diese System.

# Vertex Array Object

VAO ermöglicht uns, mehrere Primitive in einem einzigen Aufruf zu rendern, indem wir mehrere Vertex-Informationen als Array in einem einzigen Funktionsaufruf eingeben.
